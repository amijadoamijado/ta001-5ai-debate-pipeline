# 対話型解決法トリガー（提案専用）

## 目的

エラー修正が収束しない状況を検出し、対話型解決法への移行を**提案する**。
このスキルは**実行しない**。提案のみ。

## トリガー条件

以下のいずれかに該当する場合に発動：

- 同じエラーパターンが2回以上発生
- 修正を試みても問題が解決しない
- ユーザーが「迷走している」「収束しない」「同じエラーが出る」と言及

## 動作

**AskUserQuestionで提案する：**

```
質問: 対話型解決法に移行しますか？
説明: エラー修正が収束していないようです。対話型解決法では、AIの思い込みを可視化し、仕様・実装との差異を段階的に検出します。
選択肢:
- はい、/dialogue-resolution を実行する
- いいえ、現在のアプローチを継続する
```

## 重要

- **このスキルは提案のみ**
- 実際の解決プロセスは `/dialogue-resolution` コマンドで実行
- ユーザーの承認なく対話型解決法を開始しない
