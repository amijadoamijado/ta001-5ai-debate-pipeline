# Type B: 既存事業改善・収益性向上 リサーチ指示

## 案件タイプ概要
この案件は「既存事業の改善・収益性向上」を目的としています。同業他社の成功事例（ベストプラクティス）、コスト削減余地、および顧客生涯価値（LTV）の向上策が重点ポイントです。

## AI別リサーチ指示

### Claude（マーケットストラテジスト）
同業他社の収益改善事例、ビジネスモデル変革パターン、および顧客LTV向上戦略を重点的にリサーチしてください。

### ChatGPT（マーケットアナリスト）
業界平均収益指標・原価構造・価格戦略データ、および技術導入によるROI事例を重点的にリサーチしてください。

### Gemini（マーケットスカウト）
税制改正・補助金情報、市場環境変化要因、および海外同業の収益モデルを重点的にリサーチしてください。

### Grok（ソーシャルリスニングアナリスト）
競合ブランドへの不満・乗り換え意向、および業界の価格感度に関する生の声を重点的にリサーチしてください。

### Perplexity（リサーチバリデーター）
収益指標データの出典検証、および補助金情報の最新性を重点的にリサーチしてください。

### Codex（テクニカルリサーチャー）
改善施策に必要なシステム改修の技術的難易度、および既存システムとの統合可能性を重点的にリサーチしてください。
