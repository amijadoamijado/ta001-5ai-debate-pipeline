## How to Use This Index

### Adding a New Bug

1. Get the current `Next SEQ` for your project
2. Create bug report: `BUG-{PROJECT}-{SEQ}-{TIMESTAMP}.md`
3. Add entry to Search Index
4. Increment the `Next SEQ` counter
5. Add to appropriate Category section

### Searching for Related Bugs

1. Check the Keywords column for matching terms
2. Browse Categories for similar issue types
3. Reference related bugs in your new report
